# Create Your First Deck

MikroSlides opens directly into a deck editor. The top bar manages deck files and presentation actions, the left rail manages slides, the canvas edits the current slide, and the right panel changes slide or object settings.

## Start From Scratch

1. Rename the deck in the top bar.
2. Use **+ Slide** in the left rail to add slides.
3. Pick a layout from the bottom toolbar when you want a ready-made starting point.
4. Add text, shapes, or images from the toolbar.
5. Write speaker notes in the slide panel when the slide itself should stay clean.

Decks autosave to the browser as you work.

## Start From An Outline

Use **Deck from outline** when you already have notes. Paste Markdown with headings and bullets, then create the deck. MikroSlides turns sections into slides and chooses simple layouts based on the content.

A good outline looks like this:

```markdown
# Launch Plan

## Why now
- Customer demand
- Team readiness
- Market timing

## Rollout
- Beta
- Public launch
- Measure

Notes for the presenter can live as normal paragraphs.
```

After import, review each slide, adjust layouts, and run **Polish deck** if the deck needs cleaner spacing and type scale.

## Save A Slide As A Template

When a slide has a layout you want to reuse, choose **Save slide as template** from the toolbar. Saved templates stay local to the browser and appear with the built-in layouts.

Templates are useful for recurring section slides, branded layouts, or a repeated chart/text structure.
