title: Your First Deck
description: Start a deck from scratch or from Markdown.
---

MikroSlides opens directly into the deck editor. The top bar manages deck files and presentation actions, the left rail manages slides, the canvas edits the current slide, and the right panel changes slide or object settings.

## Start From Scratch

1. Rename the deck in the top bar.
2. Use **+ Slide** in the left rail to add slides.
3. Pick a layout from the bottom toolbar when you want a structured starting point.
4. Add text, shapes, or images from the toolbar.
5. Write speaker notes in the slide panel when the slide itself should stay clean.

Decks autosave to the browser as you work.

## Start From Markdown

Use **Deck from Markdown** when you already have notes or generated draft content. Paste Markdown with headings, bullets, optional frontmatter, slide metadata, and speaker notes, then create the deck.

A basic Markdown deck looks like this:

```markdown
---
title: Launch Plan
aspect: 16:9
---

# Launch Plan

---
layout: bullets
## Why now
- Customer demand
- Team readiness
- Market timing

---
layout: bullets
## Rollout
- Beta
- Public launch
- Measure

notes:
Mention timing and owner.
```

After import, review each slide, adjust layouts, and refine the objects directly on the canvas.

## Save a Slide as a Template

When a slide has a layout you want to reuse, choose **Save as template** in the slide inspector. Saved templates stay local to the browser and appear with the built-in layouts.

Templates are useful for recurring section slides, branded layouts, or a repeated chart/text structure.
