MikroSlides ships as a static browser app. No account server, database, API token, or runtime secret is required.

:::tip[Use MikroSlides Online]
Open [slides.mikrosuite.com](https://slides.mikrosuite.com) to use MikroSlides online for free. It runs directly in the browser over HTTPS and stores decks privately in browser storage for that site.
:::

## Prerequisites

- Node.js 24 or later
- A static file server for local testing

## Download

```bash
curl -sSL -o mikroslides.zip https://releases.mikrosuite.com/mikroslides_latest.zip
unzip mikroslides.zip -d mikroslides
```

The extracted folder contains the static app files: `index.html`, `assets/main.js`, `assets/styles.css`, and copied static assets.

## Serve Locally

```bash
cd mikroslides/*
npx http-server . -a 127.0.0.1 -p 8000 -c-1
```

The local server starts at `http://127.0.0.1:8000`.

## From Source

```bash
npm install
npm run dev:web
```

The source app server starts at `http://127.0.0.1:8000`.

## Documentation Site

```bash
npm run docs
```

The documentation server starts at `http://127.0.0.1:4321/slides/docs`.
