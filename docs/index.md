# MikroSlides Docs

MikroSlides is a local-first presentation app for focused decks. It keeps work in your browser by default and gives you explicit file exports when you want backups, handoff, or sharing.

## Product Boundaries

MikroSlides is intentionally compact. It is not a multiplayer presentation workspace, a design suite, or a cloud file manager. It focuses on direct deck editing, local storage, simple presentation playback, and portable exports.

Use it when you want to draft, refine, present, and export a deck without signing into a large product surface.
